# Lesson 2 Lab — Using Essential Tools

Complete the following tasks. Do not refer to your notes.

---

1. Create an empty file called `testfile.txt` in your home directory.

2. Write the text "Hello RHCSA" into a file called `notes.txt`, creating it in the process.

3. Append the text "Study hard" to `notes.txt`.

4. Display the contents of `notes.txt`.

5. Open the `notes.txt` file in Vim, add a new line that says "Exam ready", save, and quit.

6. Open the man page for the `ls` command and find what the `-h` flag does. Then quit the man page.

7. Search all man pages for anything related to the keyword "password".

8. Open section 5 of the man page for `passwd`.
