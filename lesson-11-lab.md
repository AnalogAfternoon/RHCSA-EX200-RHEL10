# Lesson 11 Lab — Managing Network Configuration

Complete the following tasks. Do not refer to your notes.

---

1. Display all IP addresses assigned to your system.

2. Show the current routing table.

3. Display the status of all network devices.

4. Show the current hostname of the system.

5. Set the hostname to `rhcsa-server.local` permanently.

6. Add an entry to `/etc/hosts` that maps `192.168.1.50` to `testserver.local`.

7. Display all current NetworkManager connection profiles.

8. Configure your primary network interface to use a static IP of `192.168.1.100/24` with gateway `192.168.1.1` and DNS `8.8.8.8`.

9. Bring the connection up to apply the changes.

10. Test connectivity by pinging `8.8.8.8` and then pinging `google.com` to verify DNS resolution.
