# Lesson 1 Lab — Using the Command Line

Complete the following tasks. Do not refer to your notes.

---

1. Print your current working directory to the screen.

2. List all files in the `/etc` directory using the detailed format.

3. Change into the `/var/log` directory.

4. Without changing directories, list the contents of `/tmp`.

5. Display the current logged-in username.

6. Change back to your home directory using a single command.

7. List all files in your home directory, including hidden files, in long format.
