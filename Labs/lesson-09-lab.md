# Lesson 9 Lab — Managing Users and Groups

Complete the following tasks. Do not refer to your notes.

---

1. Create a user called `admin1` with a home directory and bash as their shell.

2. Add a comment "Admin User" to the `admin1` account.

3. Create a group called `admins`.

4. Add `admin1` to the `admins` group without removing existing group memberships.

5. Display all groups `admin1` belongs to.

6. Set `admin1`'s account to expire on December 31, 2025.

7. Set a minimum password age of 7 days for `admin1`.

8. Set a warning period of 14 days before `admin1`'s password expires.

9. View the contents of `/etc/passwd` and locate `admin1`'s entry.

10. View the contents of `/etc/group` and locate the `admins` entry.
