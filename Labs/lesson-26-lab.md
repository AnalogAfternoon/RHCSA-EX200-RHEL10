# Lesson 26 Lab — Configuring Time Service

Complete the following tasks. Do not refer to your notes.

1. Display the current date, time, timezone, and NTP sync status.

2. List all available timezones that contain "America".

3. Set the system timezone to `America/New_York`.

4. Display the current date and time using the `date` command.

5. Check the status of the `chronyd` service.

6. Display the current NTP sync status and sources.

7. Force an immediate time sync.

8. Edit `/etc/chrony.conf` to add `pool.ntp.org` as an NTP server, then restart chronyd.

9. Enable NTP synchronization using `timedatectl`.

10. Verify NTP is active and the system clock is synchronized.
