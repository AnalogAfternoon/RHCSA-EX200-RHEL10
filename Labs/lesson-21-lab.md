# Lesson 21 Lab — Basic Kernel Management

Complete the following tasks. Do not refer to your notes.

1. Display the current running kernel version.

2. List all currently loaded kernel modules.

3. Display information about the `xfs` kernel module.

4. Load the `xfs` module into the kernel.

5. Remove the `xfs` module from the kernel.

6. Configure the `xfs` module to load automatically at boot.

7. Display the current value of the `net.ipv4.ip_forward` kernel parameter.

8. Enable IP forwarding temporarily (without rebooting).

9. Make IP forwarding persistent across reboots.

10. Apply kernel parameter changes from the config file without rebooting.

11. List all installed kernel versions on the system.
