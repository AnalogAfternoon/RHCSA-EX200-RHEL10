# Lesson 4 Lab — File Management

Complete the following tasks. Do not refer to your notes.

1. Create a directory called `rhcsa-practice` in your home directory.

2. Inside `rhcsa-practice`, create three subdirectories: `scripts`, `logs`, and `backups`.

3. Create an empty file called `readme.txt` inside `rhcsa-practice`.

4. Copy `readme.txt` into the `backups` directory.

5. Move `readme.txt` from `rhcsa-practice` into the `logs` directory and rename it `info.txt`.

6. Create a compressed tar archive called `backup.tar.gz` of the entire `rhcsa-practice` directory.

7. Extract the archive into `/tmp`.

8. Delete the `rhcsa-practice` directory and all its contents.
