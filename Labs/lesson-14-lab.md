# Lesson 14 Lab — Managing Processes

Complete the following tasks. Do not refer to your notes.

1. Run `sleep 500 &` in the background and note its PID.

2. Send a graceful termination signal to that process using its PID.

3. Run `sleep 500 &` again and force kill it using SIGKILL.

4. Kill all processes named `sleep` using a single command.

5. Start the `tar` command with a lower priority (nice value of 10).

6. Change the priority of a running process to nice value 5 using its PID.

7. List all available tuned profiles.

8. Display the currently active tuned profile.

9. Apply the `balanced` tuned profile.

10. List all currently logged-in users and what they are doing.
