# Lesson 28 Lab — Configuring a Firewall

Complete the following tasks. Do not refer to your notes.

1. Check whether `firewalld` is running.

2. Display all current firewall rules including services, ports, and zone.

3. Show all listening TCP and UDP ports with their associated processes.

4. Permanently allow the `http` service through the firewall.

5. Permanently allow the `https` service through the firewall.

6. Reload the firewall to apply the permanent rules.

7. Permanently allow port `8080/tcp` through the firewall.

8. List all currently allowed services in the default zone.

9. List all currently allowed ports in the default zone.

10. Remove the `http` service from the firewall permanently.

11. Add a rich rule to allow SSH access only from `192.168.1.0/24` permanently.

12. Reload the firewall and verify all rules are applied.
