# Lesson 13 Lab — Monitoring Activity

Complete the following tasks. Do not refer to your notes.

1. Display all running processes for all users.

2. Find if the `sshd` process is running.

3. Run the `sleep 300` command in the background.

4. List all background jobs in the current shell.

5. Bring the background job to the foreground.

6. Suspend the job using a keyboard shortcut, then send it back to the background.

7. Display memory usage in human-readable format.

8. Show the system uptime and load averages.

9. Open the live process monitor and sort by memory usage. Then quit.

10. Display CPU architecture information.
