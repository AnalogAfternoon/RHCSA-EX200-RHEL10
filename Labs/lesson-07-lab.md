# Lesson 7 Lab — Managing Users and Groups

Complete the following tasks. Do not refer to your notes.

1. Create a new user called `student1`.

2. Set a password for `student1`.

3. Create a new group called `studygroup`.

4. Add `student1` to `studygroup` without removing them from any existing groups.

5. Lock the `student1` account.

6. Unlock the `student1` account.

7. Set `student1`'s password to expire every 90 days.

8. Display password aging information for `student1`.

9. Delete `student1` and their home directory.

10. Delete the `studygroup` group.
