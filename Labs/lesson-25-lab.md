# Lesson 25 Lab — Managing HTTP Services

Complete the following tasks. Do not refer to your notes.

---

1. Install the Apache web server package.

2. Enable and start the `httpd` service.

3. Verify Apache is running.

4. Create a simple web page at `/var/www/html/index.html` that says "Welcome to my RHEL web server".

5. Open the firewall to allow HTTP traffic permanently.

6. Test the configuration file for syntax errors.

7. Check the SELinux context of files in `/var/www/html/`.

8. Move a file from `/tmp/mypage.html` to `/var/www/html/` and restore its correct SELinux context.

9. View the Apache error log.

10. Reload Apache to apply any configuration changes.
