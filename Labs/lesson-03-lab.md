# Lesson 3 Lab — Bash Shell Redirection

Complete the following tasks. Do not refer to your notes.

1. Run `ls /etc` and save the output to a file called `etclist.txt`.

2. Run `ls /etc` again and append the output to `etclist.txt`.

3. Run `ls /fakedir` and redirect the error message to a file called `errors.txt`.

4. Run `ls /fakedir` and discard the error message entirely.

5. Display the first 5 lines of `etclist.txt`.

6. Display the last 5 lines of `etclist.txt`.

7. Pipe the output of `ls /etc` into `less` and scroll through it.

8. Run a command that sends both standard output and error messages into a single file called `all-output.txt`.
