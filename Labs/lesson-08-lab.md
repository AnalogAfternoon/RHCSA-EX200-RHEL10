# Lesson 8 Lab — File Permissions

Complete the following tasks. Do not refer to your notes.

---

1. Create a file called `secure.txt` in your home directory.

2. Set the permissions on `secure.txt` so only the owner can read and write it (numeric method).

3. Create a script file called `run.sh` and give the owner execute permission using the symbolic method.

4. Set `secure.txt` permissions to `rw-r--r--` using the numeric method.

5. Change the owner of `secure.txt` to `root`.

6. Change the group of `secure.txt` to `root`.

7. Create a directory called `shared` and set its permissions to `rwxrwx---`.

8. View the permissions of all files in your home directory in long format.
