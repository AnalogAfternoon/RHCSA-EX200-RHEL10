# Lesson 20 Lab — Managing the Boot Procedure

Complete the following tasks. Do not refer to your notes.

---

1. Display the current default systemd target.

2. Change the default target to `multi-user.target`.

3. Switch to `rescue.target` immediately without rebooting.

4. Switch back to `multi-user.target`.

5. List all active systemd targets.

6. Display the location of the GRUB2 configuration file for a BIOS-based system.

7. After making a change to `/etc/default/grub`, regenerate the GRUB2 configuration file.

8. At the GRUB menu (during boot), describe the steps you would take to boot into `emergency.target` for this session only. (Write the steps as your answer — no command to run now.)
