# Lesson 15 Lab — Working with Systemd

Complete the following tasks. Do not refer to your notes.

1. Check the status of the `sshd` service.

2. Stop the `sshd` service.

3. Start the `sshd` service.

4. Restart the `sshd` service.

5. Enable `sshd` to start automatically at boot.

6. Verify that `sshd` is enabled.

7. List all currently active services.

8. View the unit file for the `sshd` service.

9. Mask the `bluetooth` service so it cannot be started.

10. Unmask the `bluetooth` service.

11. List all units that have failed.
