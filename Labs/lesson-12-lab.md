# Lesson 12 Lab — Managing Software

Complete the following tasks. Do not refer to your notes.

---

1. Search for a package called `vim` using dnf.

2. Install the `vim` package.

3. Display detailed information about the installed `vim` package.

4. List all installed packages on the system.

5. Find which package owns the file `/etc/passwd`.

6. List all files installed by the `bash` package.

7. List all available dnf package groups.

8. Show the dnf transaction history.

9. Remove the `vim` package.

10. List all enabled repositories on the system.
