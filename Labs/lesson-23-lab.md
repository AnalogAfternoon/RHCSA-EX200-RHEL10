# Lesson 23 Lab — Automating with Bash Scripts

Complete the following tasks. Do not refer to your notes.

1. Write a script called `hello.sh` that prints "Hello, RHCSA!" to the screen. Make it executable and run it.

2. Write a script called `args.sh` that prints the first and second arguments passed to it.

3. Write a script called `check-file.sh` that checks if `/etc/passwd` exists and prints "File exists" or "File not found".

4. Write a script called `check-root.sh` that checks if the current user is root. If yes, print "Running as root." If no, print "Not root."

5. Write a script called `counter.sh` that uses a `for` loop to print the numbers 1 through 5.

6. Write a script called `whilecount.sh` that uses a `while` loop to count down from 5 to 1, printing each number.
