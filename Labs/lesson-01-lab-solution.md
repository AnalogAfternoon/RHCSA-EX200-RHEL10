# Lesson 1 Lab — Solutions

1. Print your current working directory.
```bash
pwd
```

2. List all files in `/etc` in detailed format.
```bash
ls -l /etc
```

3. Change into `/var/log`.
```bash
cd /var/log
```

4. List contents of `/tmp` without changing directories.
```bash
ls /tmp
```

5. Display the current logged-in username.
```bash
whoami
```

6. Change back to your home directory.
```bash
cd
```

7. List all files including hidden files in long format.
```bash
ls -la ~
```
